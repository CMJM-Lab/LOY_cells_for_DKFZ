# Code repository for "Loss of the Y chromosome drives epithelial-to-mesenchymal transition through epigenetic and transcriptomic plasticity in lung adenocarcinoma" 

This repository contains the code and data for the research paper titled "Loss of the Y chromosome drives epithelial-to-mesenchymal transition through epigenetic and transcriptomic plasticity in lung adenocarcinoma"

## Overview

- `code/`: Contains scripts for data analysis, figure generation, and statistical tests.
- `data/`: Download and put the data here for the analysis.
- `figures/`: Contains the figures and plots generated by the scripts.
- `LICENSE`: This repository is licensed under the [MIT License](LICENSE).

## Reproducing the Results

To reproduce the figures and results presented in the paper, follow these steps:

1. Clone this repository:
`git clone https://git.dkfz.de/b370/lung_group/loy_cells.git`
2. Run the R scripts in the `code/` directory:
`Rscript code/analysis.R`
3. The generated figures will be saved in the `figures/` directory.
